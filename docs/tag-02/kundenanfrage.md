# Kundenanfrage — Tag 2

Die öffentliche Startseite soll einen klareren technischen Überblick anzeigen.

Als Besucher möchte ich sehen, ob die Anwendung grundsätzlich funktioniert.
Außerdem soll sichtbar sein, ob jede Backend-Variante erreichbar ist und ob die
Datenbankverbindung funktioniert.

## Sichtbare Informationen

Für jede Backend-Variante sollen diese Informationen angezeigt werden:

- Anwendung
- Backend
- Datenbankstatus
- Zeitpunkt der letzten Prüfung im Frontend

## Wichtig

Es soll weiterhin keine Admin-Funktion geben.
Es gibt noch keine Registrierung, keinen Login, keine Rollen und keine Datenbanktabellen.

Die Seite ist eine technische Startseite für den Lernfortschritt.
