# Anforderungen — Tag 1

## Funktionale Anforderungen

- Das Frontend zeigt eine öffentliche Startseite.
- Das Frontend ruft drei Backends ab.
- Jedes Backend liefert einen Status.
- Jedes Backend prüft die PostgreSQL-Verbindung.
- Das Frontend zeigt erfolgreiche Verbindungen an.
- Das Frontend zeigt Fehler an, wenn ein Backend nicht erreichbar ist.

## Nicht-funktionale Anforderungen

- Der Code ist stark kommentiert.
- Die Struktur ist erweiterbar.
- Die Dokumentation ist auf Deutsch.
- Noch keine Registrierung.
- Noch kein Login.
- Noch keine Rollen.
- Noch keine Benutzerverwaltung.
