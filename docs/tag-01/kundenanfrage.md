# Kundenanfrage — Tag 1

Der Kunde möchte ein technisches Grundgerüst für eine spätere Admin-Plattform.

Am ersten Tag soll noch keine fachliche Funktion umgesetzt werden.
Die Anwendung soll nur zeigen, ob Frontend, Backend und Datenbank erreichbar sind.

Die öffentliche Startseite soll einfach bleiben.
Später werden Änderungen aus dem Admin-Bereich Auswirkungen auf diese öffentliche Seite haben.
